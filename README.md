# VantagePay

Solana-based virtual card platform. Issue Mastercards in seconds, top up with SOL, USDT, or $VAN.

[![CI](https://github.com/pawell11/vantagepay/actions/workflows/ci.yml/badge.svg)](https://github.com/pawell11/vantagepay/actions/workflows/ci.yml)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

**Stack:** Next.js 15, Solana/Anchor, SQLite, pnpm monorepo

```bash
pnpm install
pnpm dev        # localhost:3000
pnpm test       # vitest + anchor test
pnpm lint       # eslint
```
